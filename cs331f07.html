  1. Integer Under/Over-flow: Fri. Sept. 14
    Submit a document which answers the following questions:
    • For signed and unsigned types of char, short, int, and long, print the following expressions:
      • min - 1
      • min * min
      • max + 1
      • max * max
      • temp (of same type) = min - 1
      • temp (of same type) = min * min
      • temp (of same type) = max + 1
      • temp (of same type) = max * max
    • Were there compilation errors?
    • Were there run-time exceptions?
    • How did the results differ from your expectations?
    • Which is better, using <climits> or <limits>? Why?
    • Are there any compiler switches/options that can help provide more reasonable behavior in these cases of integer under/over-flow.
    • Find and identify an external class library which can provide more reasonable behavior in these cases of integer under/over-flow.
    • Why is this an important issue?
