Introduction:

This lab implements file I/O and uses multidimensional arrays.

Lab Assignment:

Problem Motivation and Definition:

Standard I/O is useful, but sometimes limiting. Our programs can be more flexible if we are able to read and write to files directly without having to rely on the file redirection operators in UNIX.

We will modify program 8-15 in the text to read and write from files. Program 8-15 reads and writes two dimensional tables using two dimensional arrays and calculates and displays row and column averages.
The dimensions of the tables will be read from the first line of the input file.

Solution Specification:

    1. Create a new directory, lab11, and move to it.
    2. Copy ~cs201400/p15mod.cpp, ~cs201400/data15.txt, and ~cs201400/data15a.txt to your lab11 directory.
    3. Modify program p15mod.cpp to read from infile15.txt and write to outfile15.txt.
      [See Program 7-8 in the text for examples]
    4. Also, perform the following modifications:
      1. Reset MAX_ROWS and MAX_COLS to be 12 and 10 respectively.
        These will be the largest possible dimensions of our input table.
      2. Read the number of rows and columns from the first line of the input file.
        They will be ordered with the row count preceding the column count.
        Your program will use these values as the limit in the terminating expression of the respective for loops.
      3. Modify the printTables function so that the line of underscores is adjusted to match the number of columns input.
      4. Place column headers over each column. E.g., COL0 COL2 ... COLn AVG
        Place another row of underscores between the column headers and the top row of the output table.
    5. Run the program twice: once for the data in data15.txt and once for the data in data15a.txt.
      You will need to copy each input file to infile15.txt prior to running the program.
      After each run terminates, rename the output to be output15.txt and output15a.txt respectively.
    6. When correct, generate a printout of the source and the two ouput files.
