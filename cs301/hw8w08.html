Programming Assignment 8

Sorting Algorithms Comparison

Implement and compare the following sorting algorithms on input data with orderings which will affect the performance of the sort.

  1. Implement the follwing sorting algorithms:
    1. merge sort
    2. quick sort
      1. Extra Credit: median of three method for pivot
      2. Extra Credit: random pivot
    3. radix sort
    4. counting sort
  2. Test using data [size == (100 -- 1000)] with the following orderings:
    1. random
    2. forward sorted
    3. backwards sorted
    4. few digits
    5. many duplicates {having different frequencies}
  3. For each data set, count the number of times each algorithm
    1. examines a datum
    2. writes a datum
  4. Document your observations of the performance of these sorts relative to the data set. What conclusions can you draw from your observations?
