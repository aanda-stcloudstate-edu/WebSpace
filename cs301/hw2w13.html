Recursion Exercises

Write programs that demonstrate the following recursive and iterative functions.

For the following recursion exercises:

  • Create an output (text) file to submit by using the Unix script command to iteratively display each problem solution in the following order:
    1. the function source code [include the problem number as a header comment in each function. (e.g. "Problem 2.3" for the third problem of this homework)]
    2. white space
    3. a few runs of the function's test driver using good, representative, test data
      (i.e. the output should not be a palindrome, or all the same characters, if ordering is a concern)
    4. white space

We want these recursive functions to be lightweight and as terse as possible (but readable), so for each recursive function,

  1. have a header comment line stating your name, course, semester & year
  2. have a header comment line stating the problem number
  3. have a header comment line stating what the recursive function does
  4. have a header comment line for each parameter stating its preconditions
  5. have an in-line comment identifying the base case. (even if the base case is to do nothing)
  6. use assert() to enforce the preconditions.
  7. use descriptive identifiers
