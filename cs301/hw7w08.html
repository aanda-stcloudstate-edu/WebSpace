Programming Assignment 7

Implement Programming Problem 10.3: write a template version of the pointer -based Binary Search Tree (BST)

  • For this assignment, you will refactor the Carrano text's BST source.
  • Start by refactoring the binary tree (BT) code to use templates.
  • Implement BST methods via BT methods wherever appropriate.
  • Implement the following additional methods for the BT class:
    (Throw exceptions where appropriate)
    • Count number of nodes
    • Compute the tree height
    • Determine if one item is an ancestor of another
    • Find a minimum element
    • Find a maximum element
    • Find a specific element
    • Overload the == operator (both shape and values)
    • Extra Credit: Display the shape and optionally the contents
      (useful for debugging and demonstrations)
      • Print consecutive integers indicating index values
      • The root will be alone on the first row
      • Print each subsequent level from left to right on its own row
      • Optionally follow with the listing of values, one per line in index order.
  • Implement the following additional methods for the BST class:
    (Throw exceptions where appropriate)
    • Save to a file (preserving shape)
    • Restore from a file (preserving shape)
    • Treesort
    • Balance

  • Ensure that your program detects bad, undefined, and out-of-range input errors both on the command line and during execution. Error messages should go to standard error.
  • Focus on the modularity of your design. Try to maximize cohesion and minimize coupling.
  • Before the program exits, be sure to release back to the heap all dynamic memory objects.
  • Use typedef to establish the types for the exponent, the coefficient and the independent variable
