Programming Assignment 8 Testing Polynomials

Test your polynomial class by using it to generate the first ten or so polynomials from the following families of orthogonal polynomials.

The Hermite and Chebyshev polynomial recurrences will work for integer coefficients.

Hermite polynomials

Use the following recurrence relations:
Probabilists' form
\(He_{n+1}(x) = xHe_{n}(x) - nHe_{n-1}(x)\), where \(He_{0}(x) = 1\) and \(He_{1}(x) = x\)

or, using the 1st derivative,

\(He_{n+1}(x) = xHe_{n}(x) - He_{n}'(x)\)

Physicists' form
\(H_{n+1}(x) = 2xH_{n}(x) - 2nH_{n-1}(x)\), where \(H_{0}(x) = 1\) and \(H_{1}(x) = 2x\)

or, using the 1st derivative,

\(H_{n+1}(x) = 2xH_{n}(x) - H_{n}'(x)\)

The explicit sequences of these polynomials, \(He_{0}(x)\) through \(He_{10}(x)\), and \(H_{0}(x)\) through \(H_{10}(x)\), are here.

Chebyshev polynomials

Use the following recurrence relations:
first kind
\(T_{n+1}(x) = 2xT_{n}(x) - T_{n-1}(x)\), where \(T_{0}(x) = 1\) and \(T_{1}(x) = x\)
second kind
\(U_{n+1}(x) = 2xU_{n}(x) - U_{n-1}(x)\), where \(U_{0}(x) = 1\) and \(U_{1}(x) = 2x\)

The explicit sequences of these polynomials,

\(T_{0}(x)\) through \(T_{11}(x)\), and \(U_{0}(x)\) through \(U_{9}(x)\) are here.
