Lab 12

Introduction:

This lab exercises pointers and multiple levels of indirection.

Lab Assignment:

Problem Motivation and Definition:

Before we may harness the power of pointers, we must first understand and gain a facility with using multiple levels of indirection and dereference that pointers enable.

We will develop a program to implement projects 9.56, 9.57 and 9.58.

Solution Specification:

    1. Create a new directory, lab12, and move to it.
    2. Create a program, lab12.cpp, which implements projects 9.56, 9.57 and 9.58.
      Additionally, for each project, display the value of all variables (including un-dereferenced pointer variables).
      Label all output.
      Create your own data.
    3. When the program is working correctly, generate a printout of the source and the ouput.
