Lab Assignment:

  1. Create a new folder, Lab4, in your labs folder.
  2. Move into your Lab4 folder
  3. Using a syntax-highlighting text editor, open a new file to edit. Save this new file as Lab4.html.

Create a single Web page (Lab4.html) which incorporates all requirements of each of the following Exercises in Reed:
(Please separate the display of each of your three solutions by two horizontal rules.)

  1. Exercises 5.5 & Exercises 5.13
    • Perform all of the requirements that the problem specifies.
    • (This kind of interactive story is also trademarked as a MadLib™.)
    • Prompt for all missing words before you begin to display the story.
    • Use the generic term for what you are prompting for as a default value in each case.
         (e.g. the default value would be "animal" if you are prompting for an animal)
    • Additionally, prompt for at least one of each of the following:
      • noun
      • adjective (modifies a noun)
      • verb
      • adverb (modifies a verb)
    • Below the input boxes, between two horizontal lines, write the story using innerHTML and a corresponding <div> region.
    • for Exercise 5.13, you may select between your personal preference of displaying the boxes running either across the page or down the page.
  2. Exercise 5.6
    • You may reuse mystery.gif as your third image. (or use an appropriate external hyperlinked image of your own choice)
  3. Exercise 5.19 & Exercises 5.20
    • Try exploiting the *= & /= assignment operators.
    • Instead of creating a new Webpage, double.html, append these new features to the current Webpage.
    • Label the first button: "Click to Double", which will double the numerical value everytime the button is pressed.
    • Label the second button: "Click to Halve", which will halve the numerical value everytime the button is pressed.
