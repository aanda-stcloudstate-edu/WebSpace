Introduction:

This lab exercises the ability to:
  • Write, run, and test an assembly language program for Reed's Complete Knob&Switch Computer Simulator.

Lab Assignment:

  1. Create a new folder, Lab10, in your labs folder.
  2. Move into your Lab10 folder

For the following two Exercises, before programming the Simulator, type (or write down) your assembly program, then step through it.
(Hint: at first, use variables for your data memory addresses, then when your code passes your step-through, and you know the address of the halt statement, then replace the variables with the appropriate addresses)

  1. Exercise 14.13 with the following modification
    • use the memory location immediately following the HALT instruction as your (input) operand (the value to be multiplied by four.
    • use the memory location immediately following the input operand as your (output result) product destination.
    • use only one register.
  2. Create a program to perform the multiplication operation by computing the product of the positive integers in the first two memory locations follwing the HALT command. Store the resulting product in the third memory location following the HALT instruction.
    [HINT: create a loop for which every iteration adds one operand to a running sum
    (which will represent the product at the end) while decrementing the other operand.
    Stop the loop, then store the (resulting) product when the decremented value is zero.
    (consider using a Branching Instruction from the K & S Model 2 - Assembly & Machine Language Instruction Set, such as the BZERO instruction, to implement loops)]
  3. Extra Credit: Raise the positive value in the first of two memory locations follwing the HALT instruction to the power of the positive value in the second memory location and store the result in the third memory location.

When each simulation performs correctly, submit screenshots of the simulator, before and after each assembly language program runs on the simulator, to the Lab10 D2L dropbox.
