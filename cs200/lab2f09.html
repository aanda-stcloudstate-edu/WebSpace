Introduction:

This lab exercises the ability to:
  • Use an editor to create and modify a Web page.

Lab Assignment:

  1. Create a new directory, Lab2, in your class account directory.
  2. Move into your Lab2 directory
  3. Using a syntax-highlighting editor, open a new file to edit. Save it as Lab2.html.
  4. For each of the following tasks,
    1. modify your Web page file via the editor
    2. save the file
    3. refresh the browser open to your Web page
    4. verify that your modifications did what you've intended
Task 1 - Create a Web page
Add an <html> tag pair;
then add <head> and <body> tag pairs inside;
then add a <title> that reads Lab 2 (your name here); then add some text about yourself into the <body> region.
Task 2 - Text spacing
Add additional paragraphs to your Web page. You can write about yourself or describe your hobbies. Next, list some favorite books or movies.
Use <p>, <br>, and &nbsp; appropriately to make your page readable and attractive.
Task 3 - Headings and alignment
Divide your home page into sections. Use a large heading to list your name so that it is centered at the top of the page, followed by a horizontal line. For each subsequent text section, use a smaller heading. At least one section of your page should contain multiple paragraphs.
Task 4 - Font formatting
Augment your home page by coloring some of the text and using font formatting where appropriate. For example, if you listed favorite books or movies, you could italicize or underline their titles.
Task 5 - Hyperlinks
Add a section to your home page containing at least five hyperlinks to sites that you consider useful and/or interesting. To ensure that these hyperlinks work correctly, you should click each one and then view the linked page in your browser. For example, if you listed favorite books or movies, you could italicize or underline their titles.
Task 6 - Images
Look around the Web, and choose a public-domain image that you would like to include on your web page. You must be careful not to violate copyrights or otherwise infringe upon the rights of the owner unless explicitly stated otherwise, you should assume that any image on the Web is private property. Once located, copy the image to your LAB1 directory and incorporate the local image into your home page. Be sure to include alternate text for your image via the ALT attribute.
When finished (after you’ve proofread your Web page), submit your file to the appropriate D2L dropbox.
