Introduction:

This lab exercises the ability to:
  • Write, run, and test an assembly language program for Reed's Complete Knob&Switch Computer Simulator.

Lab Assignment:

Perform the following Exercise in Reed:
  1. Exercise 14.13 with the following modification
    • use the memory location immediately following the HALT instruction as your (input) operand.
    • use the memory location immediately following the input operand as your (output result) product destination.
  2. extra credit: Multiply together the positive integers in the first two memory locations follwing the HALT command. Store the (result) product in the third memory loacation following the HALT instruction.
    [HINT: create a loop which every iteration adds one operand to a running sum (which will represent the product at the end) while decrementing the other operand. Stop the loop and store the (result) product when the decremented value is zero.]

Note: the assembly language manual for the Complete Knob&Switch Computer Simulator is here.

When correct, submit screenshots of the simulator, before and after the your assembly language program runs on the simulator, to the appropriate D2L dropbox.
