Task 2  

Recall from Chapter 1 that computer memory is comprised of individual bits of data. A bit (short for binary digit) can store one of only two values, commonly referred to as 0 and 1. However, using two bits, you can represent four different values through the bit patterns 00, 01, 10, and 11. With the three bits, you can represent eight different values – via 000, 001, 010, 011, 100, 101, 110 and 111. In general, N bits of memory enable you to represent 2N different values.

 

Create a Web page named bits.html that prompts the user to enter a number of bits and stores that number in a variable named N. Your page should then compute and display 2N, the number of values that can be represented using the specified quantity of bits. For example, if the user entered 10 at the prompt, the page would display the message:

 

     With 10 bits, you can represent 1024 different values.

 

Once you have created your page, use it to determine the number of values that each of the following can represent:

            8 bits (1 byte)

     16 bits (2 bytes)

     32 bits (4 bytes)

     64 bits (8 bytes)
