In order to approximate the reciprocal of some number N, Newton's algorithm starts with an initial approximation of 1,
then repeatedly refines the approximation as a function of the old approximation with N.
More succinctly,

newApproximation = oldApproximation * (2 - N * oldApproximation)

To see Newton's algorithm in action, enter the number for which you want to compute the reciprocal, then click the button to see each successive approximation.
