Homework Assignment 4: Changing Bases via Loops

Rewrite your solution to Homework 2 to use loops.
The number of times you will need to loop can be computed as the number of digits of the smallest base.
The number of digits of a number N in the smaller base is 1 + log [smaller base](N)
The numbers can now be larger in value than 255.

Create a Webpage with the following elements:
