Introduction:

This lab exercises the ability to:

Lab Assignment:

  1. Create a new Webpage in a new folder, lab4, and move to it.
  2. With another partner, on paper, write an algorithm for directing an intelligent robot what to do when entering a dimly lit room. The light in the ceiling is burned out, but the robot doesn't know that. (the switch has to be thrown first. There could have been a power failure, so after the bulb is changed, it has to be tested.)
    The robot knows where in the room the following are located:
    • the light switch
    • the light bulb in the ceiling
    • the ladder to get to the ceiling bulb
    • the replacement bulb
  3. Individually, express this algorithm on your new Webpage, as an outline, using a hierarchichally ordered list.
  4. Also, on your new Webpage, write the answers to the following questions on page 153 of Reed:
