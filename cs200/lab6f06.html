Introduction:

This lab exercises the ability to Perform I/O via ONCHANGE text boxes in the context of event driven programming. We are also utilizing the two division operators to implement conversion between binary and decimal in both directions.

Lab Assignment:

Using the copybox.html [Web page / file] from the author's resource page, Use the ONCHANGE attribute to convert the binary number in one box to the decimal in the other, and vice versa.
The binary number will have to be stored as a decimal whose digits are constrained to the values of zero and one. E.g. binary five [ 101 ] is stored as decimal one hundred one.
Label the text boxes appropriately.
Assume that the data is constrained to one byte of storage so that the maximum value is a positive decimal integer between zero and 255 inclusive, and the maximum binary is 11111111.
Ensure that in your code identifiers (e.g. variable and function names) are labeled appropriately.
