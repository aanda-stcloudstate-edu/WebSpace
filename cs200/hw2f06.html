Homework Assignment 2: Changing Bases

Create a Webpage with the following elements:

  1. Standard identifying information.
    1. The file name will be Homework2.html in your CS200 directory.
    2. The title (what's displayed on the top bar of the browser window) will include:
      • Your name
      • the course
      • the assignment & number
      • the date
    3. Centered at the top of the Webpage will be the same information that is in the title.
  2. Create a text-box based base conversion facility based on your Lab 6 program
    In addition to the functionality you built into your Lab 6 program, your homework assignment will have the following additional features:
    • Each value text box will have a corresponding text box for representing the base
      A valid base will be an integer between 2 and 10 inclusive.
      (Be sure to ensure that it is an integer)
    • Alert the user with a descriptive message if the they have entered a base which violates the above precondition.
    • Alert the user with a descriptive message if the they have entered a value for conversion which is negative, not an integer, or is larger than 255_10
      (This means you have to test for an equivalent value for each valid base)

After testing thoroughly (especially the set of alerts generated when any of the preconditions are violated), submit this file to its D2L Dropbox.
