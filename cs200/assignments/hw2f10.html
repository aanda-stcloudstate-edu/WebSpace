Homework Assignment 2: event-driven programming & user-defined functions;

Purpose:
This assignment exercises the use of event driven programming and the use of user-defined functions (Chapter 7, Reed)

  1. Task A: Add the following functionality to your program for Exercise 7.18 to convert it into a restaurant diner calculator.
    • Add a textbox to enter the sales tax percentage (defaulting to the MN sales tax)
    • Add a textbox to enter the tip percentage (defaulting to 20%)
    • Separate the sales amount into two subtotal textboxes for food and alcohol, respectively (they should each default to $0.00)
    • Change the button to read: Calculate Total
    • Calculate the total to pay using the following formula:
      total = food * (1 + tip_percentage/100) + alcohol * (1 + (tip_percentage - 5%)/100) + sales_tax * (food + alcohol)
    • blur() the total
    • Label all boxes
  2. Task B: display the roll of two six-sided dice whenever a Click to Roll button is pressed.
    • Use the following formula:
      document.getElementById("die").src = "http://dave-reed.com/book/Images/die" + roll() + ".gif";
    • roll() should be a function you define to roll a random integer between 1 and 6 inclusive. ([1,6])
    • You may modify the code in dice.html
