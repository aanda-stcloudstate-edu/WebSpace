Homework Assignment 5: Complete Knob&Switch Computer Simulator assembly language programming

Introduction:

This assignment exercises the ability to:
  • Write, run, and test an assembly language program for Reed's Complete Knob&Switch Computer Simulator.
Perform the following tasks:
  1. Exercise 14.13 in Reed, with the following modifications:
    • use the memory location immediately following the HALT instruction as your (input) operand.
    • use the memory location immediately following the input operand as your (output result) product destination.
  2. (positive) integer multiplication
    1. Multiply together the positive integers in the first two memory locations follwing the HALT command.
    2. Store the resulting product in the third memory location following the HALT instruction.
    • [HINT]: create a loop in which every iteration adds one operand to a running sum (which will represent the product at completion) while decrementing the other operand.
      Stop the loop and store the (result) product when the decremented value is zero.
      You can use the BRANCH command to repeat the loop, & the BZERO command to exit the loop.

Note: the assembly language manual for the Complete Knob&Switch Computer Simulator is here.

When correct, submit text-files of your assembly language programs & screenshots of the simulator, before and after your assembly language programs run on the simulator, to the appropriate D2L dropbox.
