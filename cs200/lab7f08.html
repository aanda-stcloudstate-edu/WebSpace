Is a Point in a Box?

Introduction:

This lab exercises the ability to:
  • Work with conditionals
  • Work with coordinate geometry

Lab Assignment:

  1. Create a new Webpage in a new folder, lab7, and move to it.
  2. Determine whether a given point's (X,Y) coordinate pair is in a specified box area, using the following components:
    • Textboxes for each of the [input] values of the box coordinates: Top, Bottom, Left, and Right.
    • Alert the user to the following input errors: Top ≤ Bottom or Left ≥ Right.
    • Two labeled textboxes for each of the (X,Y) coordinates.
    • Two labeled buttons to determine whether or not the box border is considered to be in the box. (Establish and use a default value)
    • A textbox which displays the border inclusion status
    • A button to test whether or not the point is in the box
    • A textbox to display and state whether the point is in the box.

Label descriptively all boxes and buttons (a table might be in order).

Test your code.

Try to use functions to help organize and simplify your code.

When finished, submit to the dropbox.
