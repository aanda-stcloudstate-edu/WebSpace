Introduction:

This lab exercises the ability to:
  • Use an editor to create and modify a Web page.
  • Submit a Web page to a D2L dropbox

Lab Assignment:

  1. Create a new directory, Lab2, in your Labs directory.
  2. Move into your Lab2 directory
  3. Using a syntax-highlighting (e.g. gedit) editor, open a new file to edit. Save this new file as Lab2.html.
Task 1 - Create a Web page
Start by entering the language specification line: <!doctype html> Add comment lines which identify:
  • The location (full pathname) of this file on csci2
  • Your name
  • Your section number
  • The date of last edit
  • The purpose of this Web page
Below the comment lines, add an <html> tag pair;
then add <head> and <body> tag pairs inside;
then add a <title> that reads Lab 2 (your name here);
then add some text about yourself into the <body> region.
Task 2 - Text spacing
Add additional paragraphs to your Web page.
You can write about yourself or describe your hobbies.
Use <p>, <br>, and &nbsp; appropriately to make your page readable and attractive.
Task 3 - Lists
Create a two-element HTML list with the items: favorite books & favorite movies.
Then, create two sublists itemizing your respective favorite books and movies.
Task 4 - Tables
Create an HTML table where the rows are each of your classes this semester.
And, the three columns in each row are the
  1. department code
  2. course number
  3. course title
Task 5 - Headings and alignment
Divide your Web page into sections.
Use a large heading to list your name so that it is centered at the top of the page, followed by a horizontal line.
For each subsequent text section, use a smaller heading.
At least one section of your page should contain multiple paragraphs.
Task 6 - Font formatting
Augment your Web page by coloring some of the text and using font formatting where appropriate.
For example, if you listed favorite books or movies, you could italicize or underline their titles.
Task 7 - Hyperlinks
Add a section to your Web page containing at least three hyperlinks to sites that you consider useful and/or interesting.
To ensure that these hyperlinks work correctly, you should click each one and then view the linked page in your browser.
Task 8 - Backgrounds
Add a background color or image to your Web page.
Task 9 - Images
Search the Web, and choose a public-domain image that you would like to include on your web page.
You must be careful not to violate copyrights or otherwise infringe upon the rights of the owner unless explicitly stated otherwise, you should assume that any image on the Web is private property. Once located, copy the image to your LAB1 directory and incorporate the local image into your Web page. Be sure to include alternate text for your image via both the ALT & TITLE attributes.
Task 10 - Image links
Use another image as a hyperlink to a relevant external Web page.
When finished (after you’ve proofread your Web page), submit your file to the appropriate D2L dropbox.
