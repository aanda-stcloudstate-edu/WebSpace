Introduction:

This lab exercises the ability to:

  • Write, run, and test an assembly language program for Reed's Complete Knob&Switch Computer Simulator.

Lab Assignment:

For the following two exercises, before programming the simulator, write down your assembly program on paper and walk through it.
(Hint: at first, use variables for your data memory addresses, then when your code passes your walk-through, and you know the address of the halt statement, then replace the variables with the appropriate addresses)

Perform the following Exercises:

  1. Exercise 14.13 with the following modification
    • use the memory location immediately following the HALT instruction as your (input) operand.
    • use the memory location immediately following the input operand as your (output result) product destination.
    • use only one register.
  2. Multiply together the positive integers in the first two memory locations follwing the HALT command. Store the (result) product in the third memory location following the HALT instruction.
    [HINT: create a loop for which every iteration adds one operand to a running sum (which will represent the product at the end) while decrementing the other operand. Stop the loop and store the (resulting) product when the decremented value is zero. (consider using the BZERO instruction)]
  3. Extra Credit: Raise the positive value in the first of two memory locations follwing the HALT instruction to the power of the positive value in the second memory location and store in the third.
