Introduction:

This lab exercises the ability to:
  • Use an editor to create a Web page in your personal Web space on the Campus X-drive.
  • Submit a link to this Web page to the Lab1 D2L Assignment dropbox

Lab Assignment:

  1. Create a new folder, CSCI200, in your WebSpace folder.
  2. Create a new folder, Labs, in your CSCI200 folder.
  3. Create a new folder, Lab1, in your Labs folder.
  4. Move into your Lab1 directory
  5. Using a syntax-highlighting (e.g. gedit) editor, open a new file to edit. Save it as Lab1.html.
  6. Type the source code from Figure 2.17 (Demo8.html) on page 32 of Reed into Lab1.html
    • When HTML tags {<something>...</something>}occur in pairs, type the tag pair first, then insert their contents between them.
      • (This ensures that you never forget to include the closing tag)
    • Change Demo8.html to Lab1.html in the source code.
    • Add the full pathname of this file as a comment at the top of your source code.
When finished (after you’ve proofread your Web page), submit your file to the appropriate D2L dropbox.
