Lab Assignment:

Create a single Web page which incorporates all requirements of each of the following Exercises in Reed:

  1. Exercise 4.9
    • This help icon will appear at the bottom of your Web page.
  2. Exercise 4.10
    • Display any three quotations from this quotations collection when you click on the respective image.
      • Find and link to an image on the Web which is of the person each respective quote is attributed to.
  3. Exercise 4.11
    • Apply to each image from Exercise 4.10
  4. Exercise 4.13
    • Below the quotations, center an image (of your choice) that doubles the dimensions of the image ONMOUSEOVER, and restores the original size ONMOUSEOUT.
  5. Exercise 4.15
    • Change the color of the centered page heading (e.g. "Computing Quotes").
When finished (after you’ve tested and proofread each of your Web pages),
demonstrate features of your Webpage to the instructor or graduate assistant,
then submit your files to the appropriate D2L dropbox.
