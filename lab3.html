Lab 3

Introduction:

This lab will provide an introduction to compiling on eeyore and on printing the results in the ECC102 lab.

Lab Assignment:

    1. Create a new directory, lab3, and move to it.
    2. run: cp ~cs201400/lab3/hello.cpp .
      cp ~cs201400/lab3/hello1.cpp .
      cp ~cs201400/lab3/hello2.cpp .
    3. Edit the 3 files to fill in the correct information in the square brackets
    4. Compile each one: g++ filename.cpp  -o filename
    5. Each filename should be executable now.  Confirm.
    6. Run each program by typing its name.  Redirect its output to a common output file. 
      Ensure that the output of the second and third programs are appended to the output of the first program.
    7. cat together all three source files, followed by the output file into a new single file, allout
    8. Using the instructions at http://web.stcloudstate.edu/aanda/print102.html, print and hand in a copy of allout.
