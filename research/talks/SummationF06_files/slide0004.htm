Fixed precision floating point
•Physically, a floating point number is represented in computer memory as a contiguous set of bytes having a predefined internal formatting for the partitioning into three fields: sign, significand (mantissa), and exponent. •In registers, a floating point format is extended by using some combination of guard bits (guard, round, and sticky), and extension (usually 64 à 80)
