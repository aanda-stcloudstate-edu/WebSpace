    • Before taking 301:

    We covered almost all of the syllabus topics this semester except for ennumerated types(Chapter 11-10). However, There are a few other topics which you should read, either if you are continuing on to CSCI 301 or if this is your last formal C++ course. Here are the topics I suggest:
    • ennumerated type: Ch. 11-10 (a way to label a set of integers)
    • typedef: Ch. 11-11 (rename and often simplify an existing type)
    • Structure: Ch. 11-8 (a means of referring to a collection of objects such as variables and functions using a single identifier)
    • Unions: Ch. 11-9 (reinterpret a variable of one type as another -- not convert -- not a cast)
    • Strings: Ch. 14
    • Adv. File I/O: Ch. 16 (including binary files)
    • C style I/O: Appendix k (You should be able to read it even if you don't write it)
    • Bitwise Operators: Appendix H
    • Command-Line Arguments: Appendix L (useful at the very least for specifying which files your program will read and write to and from, when and where you call your program)
    • Pointers to Functions: Appendix M (can be used to pass a function as a parameter to another function)
