Useful tools:

  • source: "source .cshrc" reads and executes .cshrc from within [t]csh
  • alias: "alias text1 text2" substitutes and runs text2 whenever text1 is used as a command
  • which:  "which vi ; which vim" provides the full path name for a command. Use the full path-name to over-ride any alias-ed commands.
