Introduction:

This lab experiments with sorting algorithms.

Lab Assignment:

Problem Motivation and Definition:

Sorting is a fundamental operation performed on data. Different sorting algorithms exhibit different characteristics based on the data. This lab will explore some of them.

We will modify and run the programs for the three sorting algorithms selection, bubble, and insertion, that we studied in Chapter 8. We will modify the three programs to reverse sort order based on the status of a flag parameter, work with multiple lines of data, and count the number of comparisons and the number of data exchanges.

Solution Specification:

    1. Create a new directory, lab10, and move to it.
    2. Copy ~cs201400/p4m.cpp, ~cs201400/p6m.cpp, and ~cs201400/p8m.cpp to your lab10 directory.
    3. Edit and complete each file to solve the above problem, rename each source file and executable to reflect the type of sort it performs.
    4. For each sort program, perform the following modifications:
      1. Create a 50 element array of random integers in the range of [0,99] to sort.
      2. Modify the array printing to print 10 integers per row. [see Program 8-3 in the text for an example]
      3. Modify the sort functions to accept a parameter [a flag] which indicates whether to sort the data from smallest to largest or from largest to smallest. You will ask for and read this flag from standard input.
      4. Use two global variables to count the number of comparisons and the number of exchanges that were performed on the data. Print the result with an identifying message at the end of the program.
    5. Run each program twice: once for a forward sort of the data, and once for a backwards sort of the data.
    6. When correct, generate a printout of the source, and demonstrate, via typescript, the source and the running of the programs. Bring the printout to the next lecture for discussion.
