Timing in C++:
  • include time.h
  • wrap the following around a loop to time (changing variable names):
    • clock_t time_ffsum_start = clock(); // begin timing ffsum loop
    • clock_t time_ffsum_end = clock(); // end timing ffsum loop
  • then calculate elapsed time using the following: 
        double looptime_ffsum = static_cast(time_ffsum_end - time_ffsum_start);
	cout << "The ffsum loop required "
	     << looptime_ffsum/CLOCKS_PER_SEC
	     << " seconds\n"
	     << " (measurement granularity: " << CLOCKS_PER_SEC
	     << " of a second)\n\n";

Topic List
Each topic will describe theory, algorithms, implementations, and applications
  • FFT
  • Iterative methods for solving linear matrix equations
    {Primarily Linear Systems, secondarily EVD, SVD, and other special equations}
  • Algorithms for structured matrices
    {A taxonomy of structured matrix classes, and exploiting that structure in more assymptotically efficient algorithms for propblems ranging from matrix-vector products to solving matrix equations involving them}
  • Generative and template metaprogramming
  • Automatically tuning numerical libraries
    {Atlas, PhiPack, etc.}
  • Exploiting short vector SIMD instructions
    {e.g. MMX, SSE, AltiVec, 3DNow}
  • Matrix multiplication algorithms
    • 3M
    • Winnograd's and Strassen's (and Winnograd variant) methods
    • matrix chain (dynamic programming)
    • matrix powers using the binary algorithm
    • matrix powers using eigendecomposition (e.g. Fibonacci iterate example)
    • matrix power-vector product using eigendecomposition
    • evaluating matrix polynomials
  • Cache and TLB conscious matrix algorithms
    {recursive BLAS, FLAME, etc.}
  • Fast Multipole algorithms and data structures
