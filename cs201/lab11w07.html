Lab Assignment:

Problem Motivation and Definition:

Structures are both a method of aggregating related variables, of not necessarily the same type, under a common label.
In fact, this aggregation can be considered to be a user-defined type. The struct is now primarily a depricated legacy of C, supplanted by the more powerful class facility. However, because it is simpler to work with, we will first work with structs. The problem we will be solving is that of finding successively more accurate rational approximations to a floating point (double) value, e.g. Pi. We will represent and access the data using structures and structure syntax. Our specific goal is to mostly replicate this table of successively better rational approximations to Pi.

Solution Specification:

  1. Create a new directory, lab11, and move to it.
  2. Declare a struct containing the following:
    • int numerator
    • int denominator
    • double approximation to Pi
    • double magnitude of error of the approximation to Pi
  3. Declare a 149 element array of the above structs.
  4. Use Pi = 4 * atan(1.0) for the exact value of Pi.
  5. In each successive array element, store the next best rational approximation.
  6. Create a function that prints out the values in one of our structs to roughly resemble a row of the target table format.
    (place a slash "/" between the numerator and denominator)
    This function will receive our struct as its parameter.
  7. Loop through the array printing each row in turn.
  8. Compare your table with the reference table.
  9. Once working, refactor your program to create a new version using hierarchical structures:
    • the numerator and denominator should have their own struct and print function
    • the approximation and error should have their own struct and print function
    • the table print function should use the above two print functions
  10. When correct, generate via enscript an HTML printout of the sources, and demonstrate, via script, the sources and the running of the program data.
