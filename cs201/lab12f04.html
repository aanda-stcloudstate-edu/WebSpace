Introduction:

This lab provides experience with two-dimensional ragged arrays.

Lab Assignment:

Problem Motivation and Definition:

This lab is intended to provide experience with initializing, printing, and indexing of two-dimensional ragged arrays as well as experience with pointers and dynamic memory which are required for the implementation of ragged arrays.

We want to modify the last lab's program which initialized and printed a two-dimensional array. The array was filled with Pascal's triangle.

We will now create the two-dimensional array dynamically as as a ragged array. The problem is defined in our text as problem 9-71.

Solution Specification:

    1. Create a new directory, lab12, and move to it.
    2. Solve Problem 9-71 from page 484 of our C++ text.
      1. Read in a size for the triangle.
      2. Initialize a ragged array of the correct dimensions.

        ( See Figure 9-41 on p. 461 and Programs 9-15 through 9-18 on pages 466-468.)

      3. Note: [design decision] You have the option of building the array rows dynamically, interleaving the dynamic creation of a row with the population of its elements.
    3. Test with sizes = {1, 3, 12, 20}
    4. When correct, generate a printout of the source, and demonstrate, via typescript, the source and the running of the program for the test data.
      Note: so that the full width of the size 20 triangle may be displayed, printout using the following command rather than lpr:
      enscript --landscape filename
