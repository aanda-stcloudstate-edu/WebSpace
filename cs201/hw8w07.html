Efficient (Big-O(n log n)) Sorting

Write a program to test the following merge sort and quicksort algorithms and their specified variants.

Specifications:

  1. You will read the program data from an input file and write the sorted data to an output file (see Ch. 12).
  2. Write functions to sort in-place an array of some basic type implementing the following recursive sorting algoritms :
    1. merge sort
    2. quicksort
    Use a Boolean flag (default parameter) to indicate whether the sort is to be
    • in ascending order, or
    • in descending order.
  3. Write two new functions which refactor the previous two sorting algorithms to accept an array of pointers to the data rather than the data itself.
    The refactored sorting functions will sort the pointers by comparing the data (not the pointers themselves) but will not rearrange the data.
  4. Extra Credit: Refactor your functions to replace the Boolean flag with a Boolean comparison function duplicating the functionality of the STL sort algorithm (see your handouts).
    Test with an input set of unsorted words extracted into an array of <string>s.
    Write and use a Boolean comparison function which sorts the words:
    1. First by length ascending
    2. Second alphabetically ascending (when the same length)
  5. Before the program exits, be sure to close any open files and to release back to the heap all dynamic memory objects.
