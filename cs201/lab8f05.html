Lab Assignment:

Problem Motivation and Definition:

Knowing when to use a switch statement vs. an else-if statement is important. The switch statement, to select among several alternatives, must be used only with integral values. However, if there are ranges of many integral values, we should instead use the else-if statement.

In this lab we will solve a problem using both facilities in their appropriate respective contexts.

Solution Specification:

  1. Create a new directory, lab8, and move to it.
  2. Develop a function which implements a solution to the problem expressed in Project 53 on page 223.
  3. Your function should work for integer angles that are negative as well as for those that are greater in magnitude than 360 degrees.
  4. Develop another function to print the results.
  5. Develop and run a test driver program which validates both functions.
    • Your program will prompt the user for an input angle.
    • In addition to the test values from the text, use angles >360 and <<0.
  6. When correct, generate a printout of the source, and demonstrate, via typescript, the source and the running of the program data.
