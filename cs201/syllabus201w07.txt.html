OBJECTIVES:
    This course uses high-level language C++ and appropriate mathematical
    concepts to introduce the fundamentals of computer science, algorithm
    design, problem solving and elementary data structures and algorithms. 
    Main topics include: basic features of the UNIX operating system, editing,
    compiling, and running of C++ programs, top-down design, structure of a
    C++ program,  basic C++ constructs, standard input utput operations,
    program modularity, time complexity and analysis of algorithms, derived
    data types (pointers, arrays, structures, classes), dynamic memory
    allocation.

    

TOPICS:
    * A basic working knowledge of the UNIX operating system, including the
      use of files and an editor (vi/vim).
    * The logical structure of a digital computer.
    * An introduction to algorithms.
    * An introduction and brief survey of programming languages.
    * Entering, editing, compiling, debugging, and running C++ programs.
    * Program development and top-down design.
    * Reading and writing data: standard input and standard output.
    * Expressions and their evaluation; statements.
    * Subprograms and program modularity: functions, parameter passing,
      and default arguments; the scope of variables and functions.
    * Boolean expressions, selection statements, and DeMorgan's law.
    * Iteration (looping).
    * Reading and writing text files.
    * Time complexity and algorithm analysis.
    * Pointers and their manipulation; Passing arrays as parameters;
      memory allocation and de-allocation.
    * Enumerated types.
