Pointers (and dynamic arrays)

Write a program to rank months by their temperature variability.

Specifications:

  1. You will read the program data from a text file (see Ch. 7) which contains daily temperatures for a year (Fairbanks, AK, 1965).
    (the file uses "999" to indicate an entry for a non-existent day of the month)
    The file is on csci at [~aanda/cs201/FAI1965.txt].
  2. Store the temperatures in a ragged array with each row holding all of the temperatures for a month.
    (Note: Because you are using a ragged array, you will not be storing the 999 values.)
  3. Write a function to compute the standard deviation of the temperatures for a month and save the results in a new array.
  4. Without reordering the std. dev. array, sort the months, with respect to their standard deviation, in ascending and descending order.
    (reorder a pair of arrays of pointers)
  5. Print a pair of listings of the months (use their names not their ordinal numbers) alongside their respective standard deviations
    1. In descending order, and
    2. In ascending order.
  6. Before the program exits, be sure to close any open files and to release back to the heap all dynamic memory objects.
  7. You can check your results via a spreadsheet.
  8. Do the same for the following other statistical measures:
    • max
    • min
    • average
    • median
    • mode
