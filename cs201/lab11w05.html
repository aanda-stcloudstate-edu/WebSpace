Lab Assignment:

Problem Motivation and Definition:

In Lab 4, we read in a decimal integer but interpreted it as a binary number. We then computed and output its decimal value.
In this lab we are going to perform a sequence of two successive modifications to our Lab 4 program exploiting our knowledge of more C++ language features which which improve the readability, robustness, and generality of our Lab 4 program.

In the first section of this lab, we will modify our Lab 4 program to use an array within a loop rather than a set of related variables and code blocks. We will also make it more robust by checking the consistency of our input.

In the second section of this lab, we will further modify our program to make it more general by adding the ability to convert from a range of bases.

Solution Specification:

  1. Create a new directory, lab11, and move to it.
  2. part I: Modify a program solving lab 4.
    1. Use an array to represent the bit values.
    2. Use loops wherever possible.
    3. You may use the example program code as your base code.
    4. Ensure the consistency of the input by checking that each digit has the value of either zero or one.
      (be sure to test that you are catching bad input)
  3. Part II: Now add generality by adding the ability to convert from a range of bases.
    1. Prompt the user for a base in the range [2,10]
    2. Ensure that it is in that range.
    3. Modify the program to give the correct decimal value
  4. Part III: Now extend the range of bases to 16.
    1. You will want to use character I/O for this part to ensure the consistency of the input.
