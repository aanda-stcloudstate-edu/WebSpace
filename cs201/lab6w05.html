Introduction:

This lab provides experience defining functions, significant digits, and rounding.

Rounding (modifying a value to the nearest number having the specified number of digits, either after the decimal point or in total) is a commonly specified requirement for display of information. We quantify our certainty regarding the measurement of some quantity by indicating that we are certain about its accuracy to an estimated number of significant digits. Displaying more digits would be misleading.
Because we do not want to ever make it look like a value has more significant digits than it actually does, we lop off the all those terminal uncertain digits (while rounding) and replace them with zeros. The following functions will round a given value in both of the above ways respectively.
