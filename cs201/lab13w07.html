Introduction:

This lab provides experience with linked lists.

Lab Assignment:

Problem Motivation and Definition:

The linked list, is a recursive data structure, i.e.
one data member of the class is a pointer to an instance of the class.
One significant value of a recursive data structure is that it is efficiently dynamic. The problem we will be solving is that of:
  1. defining a list node,
  2. writing a set of methods which will together generate an ordered list,
  3. writing a method to merge two ordered lists.

Solution Specification:

  1. Create a new directory, lab13, and move to it.
  2. Define an IntNode class (see page 729)
  3. Define void headInsert() a method which inserts a node at the head of a list. (see pages 730-731)
  4. Define void insert(IntNode* afterMe) a method which inserts a node into a list after the node pointed to by the input parameter. (see page 732)
  5. Define IntNode* locateAfterMe(int target) a method which locates a node in a sorted linked list after which the target node can be inserted.
    Return the null pointer if target is smaller than the data in the first node.
  6. Combine headInsert(), insert(), and locateAfterMe() in the definition of a void insertSorted(IntNode* head, int target) method.
  7. Define IntNode* randomSorted(int size, int seed, int rangeL, int rangeR) to create a linked list of sorted random integers having size elements using insertSorted().
  8. Define void printList(IntNode* head) a method which displays, in a formatted list, the data of the linked list in order from the head.
  9. Create two random sorted linked lists of random sizes (0 ≤ size < 100).
    Display each after creation using printList().
  10. Perform Programming Project 2 on page 788.
    Display the result using printList().
