Lab Assignment:

Problem Motivation and Definition:

The array is a data structure which allows us to aggregate a set of related items of the same type under a common label. We distinguish between the items by using an index which corresponds to a sequence index.

Loops and arrays work hand-in-hand. We use loop indices in array index expressions to systematically refer to array elements.

In this lab we will use a loop to reverse the elements of an array in-place.

Solution Specification:

Create a new directory, lab10, and move to it.
Develop a reverse_array function which implements a solution to the problem expressed in Problem 29 on page 407.
  1. Input:
    1. Prompt the user to enter up to a maximum (determined by your array size) number of integers, ending with an end-of-file character.
    2. Use a loop to read those integers.
    3. Maintain a running count of the number of integers entered.
    4. If the user tries to enter more than the maximum:
      1. Discard that entry.
      2. Send the user an informative message via the standard error stream.
      3. Exit the loop.
    5. Use another loop to print out the entries, ten to a row.
  2. Reverse:
    1. Use a loop in our reverse_array function to select array elements to swap.
    2. Use the exchange() function from page 142 to perform the element swaps.
  3. Output: use a loop to print out the reversed entries, ten to a row.
  4. Testing: ensure that your program works for the boundary cases, and for even and odd numbers of entries.
  5. Improvement: You could modify your program to have a more user friendly user interface which allows the user to verify that the input was correct and, if not, specify an index and value to change. (ensure that the index is valid)
    This can be repeated as many times as necessary.
