Problem Motivation and Definition:

Exception handling refers to the special treatment in code of dealing with unusual or exceptional (not necessarily erroneous) conditions. In C++ we can partition the code for unexpected behavior away from that for expected behavior using the try, throw, and catch syntax. This partitioning allows code for each behavior to be more readable, maintainable, and reliable.
