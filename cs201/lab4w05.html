Introduction:

This lab exercises informative user communication, the two integer division operators, and compound assignment expressions.

Lab Assignment:

    1. Create a new directory, lab4, and move to it.
    2. Write  a working program which converts a binary number to decimal using the following algorithm (as expressed in Appendix D-2 of our C++ text and your lecture notes):
      1. You can use the ~aanda/cs201/source/CH03/p03-10.c++ program from page 102-103 of our C++ text as a template for this program.  Be sure to copy it over to a new file with another name.  {This is an example of code reuse}
      2. Ensure that your program has updated comment headers describing your identifying information, the date, and the purpose of the program.
      3. Prompt the user with explicit instructions on entering a binary number up to eight digits long as a contiguous sequence of ones and zeros.  The leading digit must be nonzero. {or it will be interpreted as an octal number}
      4. Read the input as if it were a decimal integer {that has only ones and zeros as digits}
      5. Store the value of each bit in its own integer variable using the following algorithm:
        1. the rightmost bit value is the remainder of dividing the input integer by 10 {use the % operator}
        2. to extract the bit to the left of it, first divide the input integer by 10 {use /}, then the bit value will be the remainder {again use the % operator}
        3. repeat until you have a value for all eight bits.
      6. Print the eight bits, in order, all on one line {use setw(1) for each.  This requires the inclusion of the iomanip library.}
      7. Declare all variables.
      8. At this point, compile and run your program to ensure that it is working correctly before we move on to the next stage.  The written bits must be in the same order as the input bits.  Use several test numbers.
      9. Use the algorithm on page 882 of Appendix D-2 of our C++ textbook to convert the binary number to decimal.
        1.  Use a variable to represent the successive powers of two and update it with the "*=" compound assignment operator.
        2. Update the running sum of powers of two (corresponding to the non-zero bits) in another variable with the "+=" compound assignment operator.
        3. Perform the above for all eight bits.
      10. Print the decimal value of the binary number on its own line which also describes to the user what the number is.
      11. Declare all variables
      12. Compile and run.  make sure the program is working correctly.  Use several test numbers.
      13. When correct, use script to generate a printout of the source followed by a demonstration of the execution of the program to submit.
        1. Use the following data to demonstrate: [11111111], [1010101], [1010].
