Introduction:

This lab examines and lists the local integer and floating point environments with respect to limits on csci (a Sparc architecture) and compares them with those on the authors' machine and on an Intel architecture.

Lab Assignment:

    1. Create a new directory, lab3, and move to it.
    2. run: cp ~aanda/cs201/source/CHAppndx/PE-0[12].c++ . {dot}
    3. run: cp ~aanda/cs201/source/CHAppndx/PE-0[12].Intel . {dot}
    4. compile: g++ PE-01.c++ -o PE-01.exe
    5. compile: g++ PE-02.c++ -o PE-02.exe
    7. Run: each executable {*.exe}
      8. saving the output in PE-01.Sparc and PE-02.Sparc respectively.
    8. Copy: each source file naming the copy PE-01.Author and PE-02.Author respectively
    9. Edit: each *.Author file so that it only contains the Authors' machine's output without any comment tokens
    10. Copy: each *.Author file to a new file with a *.Table suffix
    11. Edit: each *.Table file to have three columns, where each column contains the output for that row for the three different architectures. Add your name and today's date to the top of the file with a line of underscore characters underneath as a separator.
      (Hint:which filter can expedite this task?)
    12. Print a copy of the *.Table files together using lpr -# 2 outputfilename.
    13. Submit one copy of the *.Table files to D2L.
    14. Retain printout for your notes and exams.

Editing Suggestions:

  • run vim
  • set the following flags:
  • ":set cin sw=4" {this maintains indenting. See p. 206 of our Unix text for details}
  • ":syntax enable" {provides automatic syntax highlighting to help illuminate structure and mistakes}
