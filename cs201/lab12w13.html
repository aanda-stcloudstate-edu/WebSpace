Note: Everything in this lab assignment, unless otherwise stated, must be performed on the csci2 server.
  1. Create the directory lab12 in your labs directory, then move to it (you can use the mv or pushd command in Unix to move to a directory).
  2. Perform LAB 12.1, 12.2, 12.3, 12.4 and LAB 12.5 [Option 3].
  3. Use your editor to write the answers to the questions and exercises in Chapter 12 Lab in the file Lab12answers.txt locating it in directory lab12
    4. . Then submit the file to your lab12 dropbox in D2L.

  4. Note: for each lab program, rewrite the equivalence test expression if the second operand for a equivalence operator is a constant (literal) so that instead the first operand is the literal.
