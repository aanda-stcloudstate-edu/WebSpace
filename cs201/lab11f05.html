Lab Assignment:

Problem Motivation and Definition:

The array is a data structure which allows us to aggregate a set of related items of the same type under a common label. We distinguish between the items by using an index which corresponds to a sequence index.

Loops and arrays work hand-in-hand. We use loop indices in array index expressions to systematically refer to array elements.

In this lab we will use a loop to reverse the elements of an array in-place.
