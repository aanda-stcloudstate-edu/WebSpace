OBJECTIVES:
    This course uses high-level language C++ and appropriate mathematical
    concepts to introduce the fundamentals of computer science, algorithm
    design, problem solving and elementary data structures and algorithms. 
    Main topics include: basic features of the UNIX operating system, editing,
    compiling, and running of C++ programs, top-down design, structure of a
    C++ program,  basic C++ constructs, standard input output operations,
    program modularity, Data abstraction, elementary data structures, and
    dynamic data structures. Sorting and searching. Error handling and
    recovery. Time and space analysis of algorithms and big-O notation.
    Linked and sequential implementations of the list, stack, queue, set,
    and matrix abstract data types. Time management.

    

TOPICS:
    * A basic working knowledge of the UNIX operating system, including the
      use of files and an editor (vi/vim).
    * Introduction to C++: Program elements and structure. Variables, assignment, and arithmetic operators. (Gaddis, Chs. 1, 2)
    * Expressions and data types. (Gaddis, Ch. 3)
    * Control flow in C++: if statements. (Gaddis, Ch. 4)
    * Control flow: Loops. (Gaddis, Ch. 5)
    * Functions, parameters, returned values, and more. (Gaddis, Ch. 6)
    * Arrays. (Gaddis, Ch. 7)
    * Introduction to the Personal Software Process (PSP) (Humphrey, Chs. 1, 2), and one more chapter from Humphrey each week after its introduction.
    * Sorting and searching in arrays. (Gaddis, Ch. 8)
    * Time complexity and the big-O notation.
    * Pointers. (Gaddis, Ch. 9)
    * Characters and strings. (Gaddis, Ch. 10)
    * Records (structs) and arrays of records. (Gaddis, Ch. 11)
    * Classes. (Gaddis, Ch. 13)
    * Linked lists. (Gaddis, Ch. 17)
    * Stacks and queues. (Gaddis, Ch. 18)
