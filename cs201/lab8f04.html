Lab Assignment:

  1. Create a new directory, lab8, and move to it.
  2. Write a working program, and function quad_roots(a,b,c,r1,r2) which it calls, which requests from the user an ordered triple of floating point numbers. These will be the coefficients of the quadratic equation,
    ax2 + bx + c = 0.

    r1 and r2 will be the roots of the quadratic equation computed by the following formulae:
    1. Compute the discriminant: d = b2 - 4ac
      • if a & b are zero
        • There is no solution
        • Print a descriptive message to stderr
      • if d < 0
        • Solution is imaginary
        • Print descriptive message to stderr
      • if d = 0
        • Doubled root solution
        • r1 and r2 are -b/(2a) if a != 0
        • print a descriptive message to stderr if a == 0
      • if a = 0
        • Single solution
        • r1 and r2 are (-c/b) if b != 0
        • print descriptive message to stderr if b == 0
      • if d > 0
        • Create sign() function: sign(x) = -1 if x<0, else sign(x) = 1 {use ternary expression}
        • s = -(b + d1/2*sign(b))/2
        • r1 = s/a
        • r2 = c/s
  3. Ensure that your program and functions have updated comment headers describing your identifying information, the date, and the purpose of the program or function.
    Each user defined function should document what each parameter is and whether they are IN, OUT, or IN/OUT.
