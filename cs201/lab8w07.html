Lab Assignment:

Problem Motivation and Definition:

Pointers are an essential and powerful type with a wealth of diverse applications. It is important to understand how they work.
The following short programs will exercise and enhance your facility with them.

Solution Specification:

  1. Create a new directory, lab8, and move to it.
  2. Perform project 79.
  3. Perform project 80. In the program, do this once for char data and once for long double data. Also print out the size of one of the variables and the size of a pointer to it.
  4. Write a function that:
    • reads an arbitrary number of ints (or chars) from standard input
    • starts with a two element array off the heap.
    • if the array is filled and there is another number to add, then:
      • create an array of twice the size;
      • copy the contents of the old array into the new;
      • add the new value;
      • delete the old array.
    • return the pointer to the final array.
    • the final size will be available as a reference parameter.
    • display the array from outside the function.
  5. Perform Problem 71. Also write a function to display the triangle, accepting as parameters the pointer to it and the size.
  6. When correct, generate via enscript an HTML printout of the sources, and demonstrate, via script, the sources and the running of the program data.
