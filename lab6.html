CSCI 201

Lab 6

Introduction:

This lab provides experience with defining functions.

Lab Assignment:

    1. Create a new directory, lab6, and move to it.
    2. Write  a working program and, function which it calls, which requests a floating point number (to round) from the user, and an integer from the user to determine how many decimal places to round.
      1. You can use the expressions from Exercise 9 of Section 4-11 of your C++ text, and your lecture notes to help you develop your algorithm and functions.
      2. Ensure that your program and function has updated comment headers describing your identifying information, the date, and the purpose of the program or function. Each user defined function should document what each parameter is and whether they are IN, OUT, or IN/OUT.
      3. Use a function to Prompt the user with explicit instructions on entering the number to be rounded and the number of digits to round. The number of digits to round should be between zero and six inclusive.
      4. Write the output using another function. The output will display the original number, the number of digits to round, and the rounded value.
      5. Compile and run.  make sure the program is working correctly.  Use 1.456789 as the input number. Test with rounding to 0, 1, 2, 3, 4, 5 and 6 decimal places.
      6. When correct, generate a printout of the source, and demonstrate, via typescript, the running of the program for "checkout".
